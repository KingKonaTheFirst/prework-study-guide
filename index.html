HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • li is how you make comments and write lines
  • ul is for a block/paragraph of text
  • sections is where you combine headers and text
  • h1-h6 are markers for headers

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • git check -b branchname makes new branch and moves you to it
  • git status: checks what branch you are on
  • Step1:git add -A
  • Step 2: git commit -m "message"
  • step 3: git pull origin main
  • step 4: git push feature
  • Step 5: cry when it doesn't work then explain why to the rubber duck

JavaScript

  • console.log(); is how you make notes to the console
  • var name = "" is how you make a variable
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
